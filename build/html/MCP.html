6. MCP Service (Excel to JSON by AI Model MCP SSE/StreamableHTTP request)

English

WTSolutions的Excel转JSON是一系列工具，可将Excel转换为JSON，支持转换为扁平结构和嵌套结构JSON。它提供了”Excel转JSON”的全场景解决方案，包括Excel加载项、Web应用、开放API和企业级MCP工具：

  • Web应用：直接在Web浏览器中转换Excel到JSON。

  • Excel加载项：在Excel中转换Excel到JSON，与Excel环境无缝集成。

  • WPS加载项：在WPS中转换Excel到JSON，与WPS环境无缝集成。

  • API：通过HTTPS POST请求转换Excel到JSON。

  • MCP服务：通过AI模型MCP的SSE/流式HTTP请求转换Excel到JSON。(<– 您当前所在位置)
