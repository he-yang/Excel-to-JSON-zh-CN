4. WPS 加载项 (在WPS中将WPS表格数据转换成JSON)

英文

WTSolutions 的 Excel 转 JSON 是一系列可将 Excel 转换为 JSON 的工具，支持转换扁平 JSON 和嵌套 JSON。它提供了“Excel 转 JSON”的全场景解决方案，包括 Excel 加载项、Web 应用程序、开放 API 和企业级 MCP 工具：

  • Web 应用：直接在网页浏览器中转换 Excel 为 JSON。

  • Excel 加载项：在 Excel 中转换 Excel 为 JSON，与 Excel 环境无缝集成。

  • WPS 加载项：在 WPS 中转换 WPS表格数据 为 JSON，与 WPS 环境无缝集成。（<– 您当前所在位置）

  • API：通过 HTTPS POST 请求转换 Excel 为 JSON。

  • MCP 服务：通过 AI 模型 MCP SSE/StreamableHTTP 请求转换 Excel 为 JSON。

4.1 运行要求

  • WPS Windows 最新版

  • WPS Linux 最新版

4.2 获取方式

  • 获取 WPS 加载项（在 WPS 中侧载）需完成若干步骤

  • 详细步骤请访问 https://wps-workbook-to-json.wtsolutions.cn/publish.html

4.3 使用方法

  • 在“Excel to JSON by WTSolutions WPS 加载项”选项卡下，点击带有 Excel to JSON 图标的“Launch Excel to JSON by WTSolutions WPS addin”按钮

  • 选择数据源

    • 直接从 WPS 工作表中手动选择数据

  • 设置转换选项

  • 点击“Go”按钮

  • 稍后您将在“开始”按钮下方看到转换后的 JSON。您可以通过以下几种方式将生成的 JSON 保存到本地计算机：

    • 复制粘贴。JSON 生成后，会显示在文本区域中，您可以直接复制并粘贴到任意位置。

    • 复制到剪贴板。JSON 生成后，点击“复制到剪贴板”按钮，JSON 将被复制到剪贴板。

    • 保存到文件。JSON 生成后，点击“另存为”按钮，系统将提示您将 JSON 保存为文件。
