# MCP Service (Excel to JSON by AI Model MCP SSE/StreamableHTTP request)

[English](https://excel-to-json.wtsolutions.cn/en/latest/MCP.html)

WTSolutions的Excel转JSON是一系列工具，可将Excel转换为JSON，支持转换为扁平结构和嵌套结构JSON。它提供了"Excel转JSON"的全场景解决方案，包括Excel加载项、Web应用、开放API和企业级MCP工具：

* 网络版方案
    * [Web应用：直接在Web浏览器中转换Excel到JSON。](WebApp.md)
    * [Excel加载项：在Excel中转换Excel到JSON，与Excel环境无缝集成。](ExcelAddIn.md)
    * [WPS加载项：在WPS中转换Excel到JSON，与WPS环境无缝集成。](WPSAddIn.md)
    * [API：通过HTTPS POST请求转换Excel到JSON。](API.md)
    * <mark>MCP服务：通过AI模型MCP请求转换Excel到JSON。</mark>(<-- 您当前所在位置)
* 本地化方案
    * [本地化应用：在本地安装Excel转JSON应用，支持离线转换。](LocalApp.md)
    * [本地化Excel加载项：在Excel中转换Excel到JSON，与Excel环境无缝集成, 支持离线转换。](LocalExcelAddin.md)
    * [本地化WPS加载项：在WPS中转换Excel到JSON，与WPS环境无缝集成, 支持离线转换。](LocalWPSAddin.md)
