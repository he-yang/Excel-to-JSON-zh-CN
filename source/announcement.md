# 公告：Excel-to-JSON 功能调整通知

尊敬的 Excel-to-JSON 用户：

您好！为了更好地满足用户需求，提供更优质的服务，我们决定对 Excel-to-JSON 的免费版和付费版功能范围进行调整。以下是具体变化：

## 功能调整内容

### 免费版调整
- **功能扩展**：免费版现在可以使用所有之前仅限付费版的功能，包括但不限于：
  - 转换所有可见工作表
  - 转换所有工作表
  - 首列作为表头
  - 嵌套 JSON 结构转换
  - 多种嵌套 JSON 键分隔符（点号、下划线、双下划线、正斜杠）
  - 空单元格转换为 JSON null
  - 空单元格从 JSON 中排除
  - 布尔值转换为字符串 "true"/"false"
  - 布尔值转换为数字 1/0
  - 日期格式转换为 ISO 8601 字符串
  - 输出格式为 2D 数组
  - 单对象 JSON 输出为对象 {}
- **使用限制**：免费版单次最多转换 6 行数据

### 付费版调整
- **功能保持**：付费版仍然可以使用所有高级功能
- **使用限制**：付费版无行数限制，可以转换任意数量的数据
- **其他权益**：付费版用户仍然享受无广告体验

## 生效时间

本次调整将于 2026 年 3 月 14 日正式生效。

## 对用户的影响

- **免费版用户**：您将能够使用更多高级功能，但单次转换的数据行数将限制为 6 行
- **付费版用户**：您的使用体验将保持不变，仍然可以无限制地使用所有功能
