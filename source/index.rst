Excel-to-JSON by WTSolutions, Excel 插件, 在 Excel 中将 Excel 转换为 JSON（网页在线版，Windows，Mac，Linux）
=======================================================================================================================================================================
Excel to JSON by WTSolutions 是一个 **Microsoft Excel 插件** 或 **在线应用**，可以在 Excel 中将 Excel 转换为 JSON。使用 Excel-to-JSON by WTSolutions可以转换扁平和嵌套的 JSON 格式。

This is Simplified Chinese version of the documentation, refer to English version https://excel-to-json.wtsolutions.cn/en/latest/
